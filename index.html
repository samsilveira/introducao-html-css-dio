Desafio - HTML na DIO._

Este projeto tem o intuito de concluir a aula de Introdução ao HTML da DIO.me.

Esta página contém a explicação de algumas tags explicadas em aula.

Tags Estruturais

Nesta parte teremos as tags usadas para montar uma página em HTML5

Tag !DOCTYPE html
Este elemento é usado para informar ao navegador a versão do HTML que está sendo utilizada.
Tag html
Esta tag informa o início do HTML e possui sua tag de fechamento (/html).
Tag head
Dentro dessa tag temos informações como metadados, definição de carcteres utilizados na página, título da página (tag title), conexão de folhas de estilo (arquivos .CSS) dentre outros elementos.
Tag body
Dentro deste elementos estarão as partes visíveis ao usuário, será a estrutura do site propriamente dita.

Tags de Texto

Aqui teremos as tags utilizadas para a escrita de textos e suas formatações nativas

Tags h1 a h6
Este elemento define um "heading", um título de texto. Vai do h1 até o h6, reduzindo seu tamanho a medida que o número aumenta.
Para fins de boas práticas, uma página deve conter apenas uma tag h1, sendo considerada o título principal, a tag h2 sendo considerada o(s) subtítulo(s) e assim por diante.
Tag p
Esta tag define um parágrafo. É utilizado para a escrita de textos explicativos e outros tipos.
Tag br
Define uma quebra de linha dentro do parágrafo
Tag pre
Define um texto pré-formatado
Tag b
Define um texto em negrito
Tag strong
Define um texto com importante, definindo-o também em negrito
Tag i
Define um texto como itálico
Tag em
Dá ênfase em um texto, também definindo-o em itálico
Tag mark
Marca um texto, como um marca-textos
Tag small
Reduz o tamanho do texto
Tag del
Coloca o texto como se tivesse sido apagado
Tag ins
Define como um texto inserido, deixando-o sublinhado
Tag sub
Coloca o texto subescrito
Tag sup
Coloca o texto sobrescrito
Tag blockquote
Define um bloco de citação
Tag q
Define uma citação curta
Tag abbr
Define uma abreviação ou um acrônimo e sua explicação, como HTML, CSS
Tag address
Define um endereço
Tag cite
Define uma citação de obra criativa, como a pintura O Grito
Tag bdo
Esta tag pode mudar a direção de um texto. Exemplo: Exemplo
Tag code
Define que o trecho é um código

Tags de Lista

Aqui são apresentados alguns tipos de listas que podem ser utilizados e algumas tags adicionais.

Tag ul
Define uma lista não ordenada (como esta)
Tag ol
Define uma lista ordenada por números. A ordenação pode ser alterada através de atributos adicionais. Ex.: 1. 2. 3. ou I. II. III.
Tag li
Define um item da lista
Tag dl
Define uma lista descritiva
Tag dt
Define um termo na lista descritiva
Tag dd
Define a descrição do termo

Tags de Tabela

Neste tópico são apresentadas as tags usadas em tabelas.

Tag table
Define o início e o fim da tabela
Tag th
Define a célula de cabeçalho da tabela
Tag tr
Define a linha da tabela
Tag td
Define a célula da tabela
Tag thead
Agrupa as células definidas como cabeçalho
Tag tbody
Agrupa as células do corpo da tabela
Tag tfoot
Agrupa as células do rodapé da tabela
